# Quran Notes

Personal notes, reflections, and tafseer study on the Quran — published as a digital garden using [Quartz](https://quartz.jzhao.xyz/).

🌐 **Live site:** [quran.mhmz.dev](https://quran.mhmz.dev)

## About

This is my ongoing effort to document what I learn while studying the Quran — covering topics, surahs, tafseer, and thematic reflections. Notes are written in Obsidian and selectively published here.

Only notes explicitly marked with `publish: true` in their frontmatter are made public. Everything else stays private.

## Stack

- **Notes:** Obsidian
- **Publishing:** [Quartz v4](https://quartz.jzhao.xyz/)
- **Hosting:** Cloudflare Pages
